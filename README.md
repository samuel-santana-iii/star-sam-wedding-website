# starandsam.com

Static wedding website for Star and Sam, archived from the original WordPress site.

**Live site:** https://starandsam.com

## Structure

- `index.html` — home page
- `getting-there.html`, `our-story.html`, `registry.html`, `rsvp.html`
- `es/` — Spanish versions of all pages
- `assets/` — images, CSS, JS

## Deployment

Hosted on Cloudflare Pages. Pushing to `main` triggers an automatic deploy.
